# 博客网站

运行程序步骤：
```
git clone https://github.com/PongLyon/WebSite.git
cd WebSite
rails s
# 在浏览器地址栏输入
http://localhost:3000/note # 当前博客首页
http://localhost:3000/note/documents # 文章分类页面,还需完善!!!
http://localhost:3000/note/documents/15ce13b9e8d36381b12cd8f381d43536 # 第一篇文章
```
