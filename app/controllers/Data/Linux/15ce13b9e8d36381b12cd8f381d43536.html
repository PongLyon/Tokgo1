一、安装gcc

// su命令登陆root用户, 使用yum安装
yum -y install gcc gcc-c++ kernel-devel //安装gcc、c++编译器以及内核文件 
service network restart // 重启网络服务
// 等待安装完成，验证是否安装成功
gcc -v
  • 错误信息记录： 1、没有启用网络连接安装gcc时提示 ： could not retrieve mirrorlist ...... 解决方案： 
ping 127.0.0.1 // 正常
ping www.baidu.com // 错误
// 没有启用联网功能;
vi /etc/sysconfig/network-scripts/ifcfg-ens33
// 修改配置ONBOOT=no----->ONBOOT=yes
// wq保存退出

二、创建第一个C文件并运行

touch test.c // 创建c文件
vi test.c // 编辑c文件并保存退出

// 编译并运行
// 输入 gcc -o 文件名(无后缀) 文件名(有后缀) 并回车
gcc -0 test test.c
